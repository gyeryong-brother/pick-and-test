# 주식

- [x] 주식의 이름 혹은 티커를 통해 검색할 수 있다
    - [x] 이름 혹은 티커의 일부분으로도 검색할 수 있다
- [ ] 나열된 주식들을 볼 수 있다
- [ ] 주식의 상세 정보를 볼 수 있다
    - [ ] 주식의 주가, 시총의 추이를 일단위로 볼 수 있다
    - [ ] 주식의 배당, 영업이익, 순이익, 매출의 추이를 연단위로 볼 수 있다
- [ ] 주식을 관심주식으로 설정할 수 있다

# 포트폴리오

- [ ] 포트폴리오에 주식을 추가할 수 있다
- [ ] 주식의 비율을 설정할 수 있다

# 백테스트

- [ ] 투자전략을 설정할 수 있다
    - [ ] 초기 투자 자금을 설정할 수 있다
    - [ ] 리밸런싱 주기를 설정할 수 있다
    - [ ] 배당재투자 여부를 설정할 수 있다
    - [ ] 투자 시작일을 설정할 수 있다
    - [ ] 적립식 투자 여부를 설정할 수 있다
        - [ ] 적립식 투자 자금을 설정할 수 있다
- [ ] 포트폴리오를 백테스트할 수 있다
    - [ ] 가격 변화 추이를 일단위로 확인할 수 있다
    - [ ] 배당 추이를 연단위로 확인할 수 있다
    - [ ] 하락률을 일단위로 확인할 수 있다
    - [ ] 수익률을 연단위로 확인할 수 있다
